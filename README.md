# 🚀 Job Application Tracker

A modern full-stack Job Application Tracker built to simplify and organize the job search process. The application enables users to manage job applications efficiently using a Kanban-style dashboard, monitor application progress, and stay organized throughout the hiring journey.

## ✨ Features

- Secure user authentication
- Responsive dashboard built with Next.js
- Kanban board for managing application stages
- Drag-and-drop job cards
- Create, edit, and delete job applications
- Company, role, salary, location & notes management
- Search and filter applications
- Real-time UI updates
- Mobile-friendly interface

## 📂 Project Overview

The application helps users organize their job search by categorizing applications into different hiring stages.

Default workflow includes:

- Wishlist
- Applied
- Interview
- Offer
- Rejected

Users can easily drag applications between columns while maintaining all job details.

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/job-application-tracker.git
```

Install dependencies

```bash
npm install
```

Create an environment file

```env
MONGODB_URI=
BETTER_AUTH_SECRET=
BETTER_AUTH_URL=
```

Run the application

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
